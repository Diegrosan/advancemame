DOCUMENTATION HOWTO
	This is a short guide to write AdvanceMAME documentation.

	1) Use an editor which keeps the TAB chars.

	For DOS and Linux the suggested choice is `setedit' :

		http://setedit.sourceforge.net/

	with the `colored' tab enabled.

	In Windows you can use UltraEdit.

	2) Use the current format:

-----------------------------------------------------------------------------
SECTION TITLE
    Subtitle
	Description text text text text text text text text text text
	text text text text text text text.

	Options:
		item - Description text text text text text text text
			text text text text text.
		item2 - Description text text text text.

	Other text.

NEXT SECTION TITLE
-----------------------------------------------------------------------------

	The "SECTION" starts a column 1
	The "Subtitle" starts after 4 spaces.
	The "Text" starts after 1 tab
	The "item" starts after 2 tab

	3) Wrap at column 72.

	4) Add a new line:

	YYYY/MM/DD - Your Name - Description of the changes

	in the VERSION section.

	5) Before doing any changes to a file post a message in the
	devel list to inform all the developers.
	When you have finished post the modified file in the same list.

	The devel list address is :

		advancemame-devel@lists.sourceforge.net
